14 — Cursor Customizado

Cursor editorial

Cursor personalizado: ponto sólido central + anel com lag de seguimento. Desativado em mobile.

Estado padrão

#cd: 5×5px · white · solid
#cr: 32×32px · border 1px white/30

Hover em link/button

#cr: 52×52px (expand)
border-color: primary
bg: primary/8%
Lag do anel: rx += (mx - rx) * 0.12 por frame via requestAnimationFrame
Trigger: body:has(a:hover) #cr, body:has(button:hover) #cr { width:52px; height:52px }
Mobile: cursor:auto; #cd, #cr { display:none } em max-width:900px
